# Azure Logic Apps Standard EDI Demo

Azure Logic Apps Standard EDI demonstration with AS2 and X12 processing.

## Overview

This repository contains a demonstration of business-document exchange over AS2 with X12 EDI translation, orchestrated by Azure Logic Apps Standard.

## Repository Structure

- `infra/` — Bicep templates for Azure resource provisioning
- `logicapps/` — Logic App workflow definitions
- `docs/` — Documentation and specifications
- `.github/workflows/` — CI/CD pipelines

## Prerequisites

- Azure subscription
- Azure CLI
- Bicep CLI
- Git
